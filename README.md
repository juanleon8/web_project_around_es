# Tripleten web_project_around_es

Around The U.S. – Proyecto Interactivo de Tarjetas

## Descripción del proyecto

Around The U.S. es una aplicación web interactiva donde el usuario puede ver, agregar, eliminar y dar “me gusta” a tarjetas de lugares, así como editar su perfil mediante ventanas emergentes (modales) totalmente funcionales.
El proyecto está construido siguiendo buenas prácticas de HTML, CSS y JavaScript, utilizando DOM manipulation, template cloning y listeners dinámicos.

## Funcionalidades

# Perfil

Editar nombre y descripción del usuario mediante un modal.

Guardado automático en la interfaz.

# Tarjetas

Agregar nuevas tarjetas mediante un formulario modal.

# Cada tarjeta incluye:

Imagen
Título
Botón de eliminar
Botón de “me gusta” (toggle)

# Las tarjetas se generan de manera dinámica usando <template>.

# Vista de imagen

Al hacer clic en la imagen de una tarjeta:
Se abre un modal con vista ampliada.
Se muestra su título como caption.

# Eliminar tarjeta

Las tarjetas pueden eliminarse del DOM en tiempo real.

## Tecnologías usadas

HTML5 – estructura semántica y templates.

CSS3 – estilos responsivos y diseño adaptado.

JavaScript (ES6+) – manipulación del DOM, event listeners, modales, clonación de templates.
